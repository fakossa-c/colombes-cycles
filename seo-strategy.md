## 2. Structure des URLs

```
/                                    → Accueil
/nos-velos                           → Nos Vélos (catégorie)
/nos-velos/velos-de-ville            → Vélos de ville
/nos-velos/velos-electriques         → Vélos électriques
/nos-velos/vtt                       → VTT
/nos-velos/velos-enfants             → Vélos enfants
/nos-velos/accessoires               → Accessoires
/reparations                         → Réparations & Entretien
/a-propos                            → À propos
/contact                             → Contact
/blog                                → Blog / Conseils
/blog/[slug-article]                 → Articles individuels
/mentions-legales                    → Mentions légales
```

**Règles d'URL :**
- Tout en minuscules, sans accents (sauf `/a-propos` acceptable)
- Séparateurs : tirets `-` uniquement, jamais underscores ni espaces
- Pas de trailing slash (ou toujours avec — choisir et s'y tenir, avec canonical)
- Éviter les paramètres d'URL pour les pages de catégories

## 9. Stratégie de liens internes (Maillage)

### 9.1 Principe directeur

Chaque page doit être accessible en maximum 2 clics depuis la page d'accueil. Les pages à fort potentiel commercial (`/nos-velos/velos-electriques`, `/reparations`) doivent recevoir le plus de liens internes.

| Page source | Liens vers |
|-------------|-----------|
| Accueil | Nos Vélos, Vélos électriques, Réparations, À propos, Blog |
| Nos Vélos | Vélos de ville, Vélos électriques, VTT, Vélos enfants, Accessoires, Réparations |

| Vélos enfants | Accessoires (casques), Contact |
| Accessoires | Nos Vélos, Réparations |
| Réparations | Nos Vélos, Contact, Blog (révision, crevaison) |
| À propos | Contact, Nos Vélos, Réparations |
| Contact | Réparations, Nos Vélos |
| Blog (article) | Page catégorie pertinente + Réparations + Contact |

### 9.3 Ancres recommandées (texte du lien)

Utiliser des ancres descriptives avec mots-clés :
- ✅ "Découvrez nos vélos électriques à Colombes"
- ✅ "Prendre rendez-vous à l'atelier"
- ✅ "En savoir plus sur la réparation vélo"
- ❌ "Cliquez ici" (ancre générique, sans valeur SEO)
- ❌ "En savoir plus" seul (trop générique)

### 9.4 Fil d'Ariane (Breadcrumb)

Implémenter sur toutes les pages sauf Accueil :

```

Accueil > Réparations
Accueil > Blog > [Titre de l'article]
```

Le breadcrumb doit être cliquable et correspondre aux URL réelles. Coupler avec le Schema.org BreadcrumbList (section 5.4).

### 9.5 CTA récurrents dans le contenu

Définir 3-4 blocs CTA réutilisables et les insérer sur chaque page :
- Bloc "Atelier" → lien `/reparations`
- Bloc "Nos Vélos" → lien `/nos-velos`
- Bloc "Nous Contacter" → lien `/contact`
